# Getting started

## Installation

> Clone this repository: `git clone https://github.com/TPN-Labs/keez-node`

### Open the directory and run the script line:

```bash
cd keez-node 
```
```bash
npm i  # or yarn
```
```bash
rm -rf .git && git init && git add . && git commit -m "Initial commit" #Optional
```
